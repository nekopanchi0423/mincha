# mincha
ログイン機能が組み込まれた匿名掲示板サイトです。  
ユーザ登録を行うとカテゴリの作成やコメントの投稿が行えます。  
作成したカテゴリやコメントは編集・削除が可能です。  
このWebアプリはherokuで公開しています。
